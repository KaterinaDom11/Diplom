# План тестирования 
## Введение 
Документ описывает методы и подходы к тестированию, которые будут использоваться тестировщиками для тестирования приложения.

Объект тестирования — это деятельность, направленная на проверку работоспособности приложения в операциой системах Android.

## Тест-план проекта преследует следующие цели:
* Определить существующую информацию о проекте и программных компонентах,
подлежащих тестированию.
* Описать стратегии тестирования, которые будут использоваться.
* Определить необходимые ресурсы для проведения работ по тестированию.
* Привести результаты тестирования.
Результаты будут оформлены в виде отчетов. Все найденные ошибки будут
зафиксированы в баг-репортах.

## Рамки проекта
1. Форма авторизации
2.  Просмотр главной страницы "Новости"

## План работы
1. Подготовка плана тестирования.
2. Подготовка чек-листа.
3. Подготовка тест-кейсов.
4. Ручное функциональное тестирование и отчеты об ошибках.
5. Автоматизированное тестирование и отчеты об ошибках.
6. Подготовка финального отчета.
